# 🎬 COMMENCER ICI

## Pour les Impatients ⚡

Si vous ne lisez qu'une chose, c'est celle-ci !

### Étape 1 : Lancer
```bash
docker-compose up
```

### Étape 2 : Attendre
Attendez que Docker affiche "✅ Services démarrés"

### Étape 3 : Accéder
- Frontend : http://localhost:3000
- Backend : http://localhost:3001

### Étape 4 : Tester
1. Créez un compte (email, username, password)
2. Connectez-vous
3. Uploadez un fichier
4. Téléchargez-le

**Voilà !** 🎉
